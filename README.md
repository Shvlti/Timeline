# Timeline Slider

Адаптивный интерактивный таймлайн-слайдер с анимацией по кругу (GSAP), кастомной логикой навигации и отзывчивым дизайном.
Deploy: https://timeline-ashen-psi.vercel.app/

## Стек

- React + TypeScript
- GSAP
- SCSS
- WebPack
- Swiper

## Возможности

- Круговой слайдер с точками-навигаторами
- Плавная анимация вращения круга
- Активная точка меняет цвет и размер
- Двойная навигация (кнопки / клики по точкам)
- Центральный блок с датами синхронизирован со слайдом
- Полностью адаптивная версия
- Математические вычисления координат точек

## Установка

- npm install
- npm run dev

## Архитектура

- Компоненты разделены по ролям
- Математика вынесена в отдельный файл
- Логика — в кастомных хуках
- Анимации вынесены в отдельный модуль
